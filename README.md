# Ideas Board
# Jon Nairn September 2017

Assumes:
- node/yarn installed and added to path ('node -v' to check node, 'yarn --version' to check yarn)

To install/run:
1. Navigate to ideas-board-app folder
> cd ./ideas-board-app 

2. Install packages
> yarn

3. Start yarn
> yarn start

4. Navigate to http://localhost:3000 in web browser
